# Changelog

All notable updates to this Seedance 2.5 API pricing tracker will be documented in this file.

This project tracks public pricing availability only. Unpublished pricing should remain marked as `not_published`.

---

## 2026-06-25

### Added

* Created public Seedance 2.5 API pricing tracker repository.
* Added initial README with tracker purpose, methodology, provider watchlist, contribution rules, and disclaimer.
* Added MIT License.
* Added `data/seedance-2-5-providers.json` as the main provider watchlist dataset.
* Added metadata, methodology, status definitions, provider entries, and disclaimer to the JSON dataset.
* Added monitored providers:

  * BytePlus / Volcengine
  * OpenRouter
  * fal.ai
  * PiAPI
  * Kie.ai
  * EvoLink
* Added `sources/seedance-2-5-sources.md` to document public source checks and update rules.

### Current status

* Official Seedance 2.5 API pricing: Not published
* Third-party Seedance 2.5 provider pricing: Not published
* Seedance 2.0 pricing: Used only as a temporary baseline reference
* Live tracker: https://videoapicost.com/seedance-2-5-pricing

### Notes

This repository does not predict Seedance 2.5 pricing. All unpublished pricing remains represented as `null` in the JSON dataset.
