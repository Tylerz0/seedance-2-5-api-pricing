## Why This Exists

AI video API pricing is difficult to compare because providers may use different billing methods, including:

* price per second
* price per video
* token-based billing
* credit-based billing
* resolution-based pricing
* fast / standard / quality model tiers
* provider-dependent starting prices

This project is designed to help developers and AI builders track pricing status before committing to a provider or estimating production costs.

---

## Methodology

This tracker follows a conservative update policy:

1. We only record pricing that appears on public provider pages or official documentation.
2. We mark unpublished pricing as `Not published`.
3. We separate confirmed pricing from temporary baseline estimates.
4. We keep provider status and pricing status separate.
5. We link to the live tracker for the latest table and source notes.

If pricing is unclear, provider-dependent, or only shown as a starting price, it should be marked clearly rather than treated as a fixed rate.

## Contribute

If you find a public Seedance 2.5 provider listing, official pricing page, or API documentation page that is not yet reflected in the tracker, please open an issue or submit a pull request.

Useful contribution details include:

* provider name
* public pricing URL
* model ID if available
* billing unit
* resolution or tier
* whether the price is fixed, estimated, or provider-dependent
* date checked

Please do not submit unverified pricing claims.
